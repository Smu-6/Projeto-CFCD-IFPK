
# ⚔️ Compêndio RPG (Wiki)

Site multi-página com UI responsiva, modo claro/escuro, busca e acessibilidade.

## 🚀 Deploy (GitHub Pages)
1. Suba os arquivos para um repositório.
2. Vá em Settings → Pages
3. Source: branch `main` (root)
4. Acesse o link gerado.

## ✨ Recursos
- Múltiplas páginas
- Menu responsivo (mobile)
- Tema claro/escuro (persistente)
- Busca por conteúdo
- Cards animados
- Acessibilidade (skip-link, foco, navegação por teclado)

## 📁 Estrutura
```
/css
/js
/pages
index.html
README.md
```

## 🛠️ Customização
Edite os arquivos HTML dentro de `/pages` para expandir o sistema.
